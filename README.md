# pifuHD_restapi

1. Client 사용법 (Demo)
 - client.py를 실행하면 test.png를 보내고 test.obj 파일을 받는다.

2. Server 구동법
 - conda 환경설정하고 필요한 패키지를 설치한다.

 - env 파일은 추후 업로드 할 예정임

3. 구현에 참조한 라이브러리
 - https://github.com/facebookresearch/pifuhd

 - conda 환경파일 업로드 (서버 구동을 위해)
